Real league logos
-----------------
1. Drop the logo file here, e.g. nba.svg (SVG preferred; PNG also works).
2. Add its slug to manifest.json, e.g.:  ["nba","nfl"]
   (or, for a custom filename: [{"slug":"nba","file":"nba-white.svg"}])
3. Refresh. The real logo replaces the built-in emblem automatically.

Slugs: nba nfl mlb nhl premier-league champions-league ncaa ufc world-cup
White / transparent versions look best on the dark strip.
